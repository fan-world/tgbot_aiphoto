# Razdevator

Каркас партнерской мультибот-системы на `aiogram 3` + `sqlite` для image-to-video сервиса.

Что уже есть:

- админский бот с ролями `owner` и `admin`
- мультибот-раннер для персональных и общих клиентских ботов
- inline-навигация экранами через редактирование текущего сообщения
- SQLite-схема для партнеров, ботов, пользователей, платежей, генераций и рассылок
- RU/EN клиентские сценарии с каталогом шаблонов и заглушками под генерацию/оплату
- слой ассетов для бесшовного переключения экранов через `editMessageMedia`

## Быстрый старт

1. Создайте `.env` на основе `.env.example`.
2. Установите зависимости:

```bash
python3 -m pip install -e .
```

3. Запустите:

```bash
razdevator
```

## Что нужно настроить дальше

- токен админского бота
- токен общего бота либо создание общего бота через админку
- `OWNER_IDS`
- реальные платежные провайдеры для `RU` и `EN`
- реальный image-to-video API
- финальные баннеры и media id/file id при необходимости

## Архитектура

- `src/razdevator/main.py` — запуск приложения
- `src/razdevator/core` — конфиг, перечисления, модели
- `src/razdevator/db` — схема и репозитории SQLite
- `src/razdevator/services` — рантайм, экраны, генерация, платежи, шаблоны, i18n
- `src/razdevator/bot` — обработчики, middleware, состояния, клавиатуры
