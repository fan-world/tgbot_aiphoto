# Razdevator — честная бесплатная версия

Это каркас мультибот-платформы на `aiogram 3` + `sqlite`, ориентированный на
image-to-video генерицию. В этой публичной сборке платёжные провайдеры и
валютные конверсии отключены — пополнение баланса происходит мгновенно и
бесплатно (для удобства демонстрации и локального использования). Нейросеть и
логика генерации оставлены как есть (stub/интеграции в `services/generation`).

Что есть в проекте:

- Админский бот с ролями `owner` и `admin`.
- Мультибот-раннер для персональных и общих клиентских ботов.
- Inline-навигация экранами (редактирование текущего сообщения).
- SQLite-схема для ботов, пользователей, платежей (как лог), генераций и рассылок.
- RU/EN клиентские сценарии, каталог шаблонов и интеграция с генератором.
- Сервис ассетов для экранов и баннеров.

Отличие этой публичной ветки:

- Платежная логика упрощена: выбор пакета сразу начисляет кредиты пользователю
  (provider = MANUAL), валюты и внешние методы отключены.
- Поддержка нейросети/генерации оставлена — чтобы подключить реальный
  image-to-video бэкенд, настройте `services/generation`.

## Быстрый старт (локально)

1. Скопируйте `.env.example` в `.env` и укажите токены ботов и `OWNER_IDS`.
2. Установите зависимости:

```bash
python -m pip install -e .
```

3. Запустите приложение:

```bash
razdevator
```

Примечание: в этой сборке платежи не нужны — пополнение баланса в интерфейсе
происходит мгновенно.

## Подготовка к публикации в GitHub

1. Инициализируйте репозиторий и сделайте первый коммит:

```bash
git init
git add .
git commit -m "Initial public free release — payments disabled, neural parts kept"
```

2. Создайте репозиторий на GitHub и подключите remote, например:

```bash
git remote add origin https://github.com/<your-username>/razdevator.git
git branch -M main
git push -u origin main
```

3. Рекомендуется добавить `.gitignore` (если ещё нет) и проверить `LICENSE`.

## Что можно сделать дальше

- Подключить реальный image-to-video бэкенд в `src/razdevator/services/generation.py`.
- При необходимости вернуть платёжные провайдеры — код для Platega/Stripe
  присутствовал в исходной версии и может быть восстановлен.

## Архитектура (коротко)

- `src/razdevator/main.py` — запуск приложения
- `src/razdevator/core` — конфиг, перечисления, модели
- `src/razdevator/db` — схема и репозитории SQLite
- `src/razdevator/services` — рантайм, экраны, генерация, платежи (упрощено), шаблоны, i18n
- `src/razdevator/bot` — обработчики, middleware, состояния, клавиатуры
